Form-item

A form item can have text, buttons and inputs. You can have several of them in a form-item, and you just need to add 'add-on' to all but the main element (usually the input).

If you want your form-item to take up the full width, add 'expand' to both the form item, and the element inside that you want to fill up the space.

You can add all kinds of inputs to a form-item, but the implementation differs for some of them. To learn more about it, visit 'Input'.
