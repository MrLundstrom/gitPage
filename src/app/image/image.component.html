Image

The 'image' class is made to better control img elements. It can be set to a specific aspect-ratio between 1x4 - 8x4 & 1x3 - 6/3, or just let it adjust to the image.

You can't use padding/margin on the 'image' if you use a aspect ratio, so just encapsulate it in a padding/margin div

By appling the 'cover' class on the img, the image will keep it's originial aspect ratio and instead cover the image container. You can easily add a image description by adding a info class under the image container.

You can also make images rounder by using 'is-rounded' (and totally round if you set the aspect ratio to 4x4 or 3x3).
