Button

a button is an interactive component. It can be used as 'links'/'form-actions'/'actions' on a page (like toggle). a button can be used by itself, or together as a button-group or as buttons that are grouped.

You can change the apperance of a button with color classes.

It is also possible to change the regular size of a button by adding 'is-small' or 'is-large'.

You can also make a button or a button-group fill upp all available width by adding 'expand'.
