Columns

'columns' is a container for 'column' items. columns is by default a flexbox-container with flex-direction row. On mobile and tablet, it sets all column-items width to 100% and they are therefore stacked vertically. By adding 'is-mobile' or 'is-tablet', the column-items keep their default behavior down to and including that size.

'column' is a flex-item that populates a 'columns' container. By default, they take up as much space as they can, and sharing it with other column-items so that all content fits. You can set a specific width for them by adding 'is-#' where # is a number between 1 and 12. You can also offset a column to the right by using 'is-offset-#' where # is a number between 1 and the size of the column, (column is-5 can use is-offset-1 to is-offset-7).
